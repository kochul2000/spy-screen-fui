# Spy Screen FUI

가상의 첩보기관 감시 화면을 구현한 FUI(Fantasy User Interface) 프로젝트입니다.

A fictional intelligence agency surveillance screen built as a FUI (Fantasy User Interface).

## Demo

https://kochul2000.github.io/spy-screen-fui/

## Features / 주요 기능

- **Global Tracking Map** — 요원 위치를 실시간 추적하는 세계 지도
- **Agent Roster** — 현장 요원 목록 및 상태 표시
- **Surveillance Feed** — 감시 카메라 화면 시뮬레이션
- **Signal Intelligence** — 신호 정보 분석 파형
- **Comms Log** — 암호화된 통신 로그
- **Target Dossier** — 대상 프로필 및 위협 등급

순수 HTML/CSS/JavaScript로 구현되어 있으며 외부 의존성이 없습니다.

Single `index.html` file — no build tools or dependencies required.
